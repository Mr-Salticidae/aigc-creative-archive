# 风格基底测试 · 4 张关键截帧 MJ Prompt

> 工具：Midjourney v8.1。目的：用最少 4 张覆盖全片视觉需求，验证视觉语言能否在同一 sref 下焊住。
> 通用尾参：`--ar 16:9 --v 8.1 --style raw`（`--style raw` 关掉自动美化，留旧终端冷质感）

## sref 两阶段锁定法

**阶段 A — 找锚**：只跑截帧①，带 `--sref random`，跑 4–6 次，挑质感最对的一张，记下其 sref 码（填入 `../sref锚定记录/`）。
**阶段 B — 复用**：把该 sref 码 + `--sw 200` 套进 ②③④ 的 `[锚定码]`。`--sw` 范围 0–1000，本片建议 150–250。

---

## 截帧① — 系统虚空 / 建立镜头（第一章）

```
archival data-space visualization, countless tiny translucent life-frames
suspended and floating in an infinite deep blue-black void, each frame a small
still of an anonymous human moment, dispersed like a surveillance sample grid,
system observing humanity from above, vast negative space, clear visual hierarchy,
extreme wide establishing shot, faint volumetric haze, desaturated grayish-white
figures, deep blue-black color grading, sparse warm amber terminal-text glints,
cold restrained archival aesthetic, near-monochrome
--ar 16:9 --v 8.1 --style raw --sref random --sw 200
--no red, neon, cyberpunk glow, colorful, text overlay, watermark, logo, blurry
```

要点：`surveillance sample grid / observing from above` 锁系统视角；`--no red` 此张严禁出红。

## 截帧② — 单人生活瞬间 · 基准镜头（第二章，全片用得最多，亦为 B 层 sref 源）

```
a lone young person sitting alone in darkness, face turned away / unseen, lit only
by the cold glow of a phone screen, scrolling job listings deep at night, isolated
inside a vast empty blue-black space, rendered as a single system data sample,
single dominant subject, large negative space, off-center composition, subject
looking away, medium-wide shot, quiet and still, desaturated grayish-white figure,
deep blue-black ambient color grading, faint warm amber UI light, surveillance
archive aesthetic, cinematic restraint, near-monochrome
--ar 16:9 --v 8.1 --style raw --sref [锚定码] --sw 200
--no red, neon, cyberpunk, colorful, facial close-up, text, watermark, logo
```

要点：`face turned away` 既不锁脸又避恐怖谷；此张定稿后即作为 B 层人物 sref 源。

## 截帧③ — 群像面孔库（第五章"大量人类面孔"）

```
a dense archival grid of many diverse human faces — elderly, children, a prisoner,
a doctor, a cleaner, an artist, ordinary office workers — each rendered as a small
low-saturation ID-style portrait floating in blue-black space, all calm and
expressionless, evenly lit like database records, system sample catalog, ordered
grid layout, flat frontal framing, no single hero, desaturated grayish-white tones,
deep blue-black color grading, faint amber index text between frames, cold clinical
archival mood, near-monochrome
--ar 16:9 --v 8.1 --style raw --sref [锚定码] --sw 200
--no red, neon, cyberpunk, colorful, hyperrealistic skin close-up, text, watermark
```

要点：刻意 `small ID-style portrait / grid` 而非 `hyperrealistic close-up`，避群脸特写恐怖谷。

## 截帧④ — 异常值红 · 配给镜头（第四章，最关键的颜色纪律测试）

```
a single human hand holding one small deep-red train ticket, the only spot of color
in an otherwise near-monochrome scene, everything else desaturated grayish-white and
deep blue-black, predawn stillness, the red ticket as a quiet anomaly, single
dominant focal point on the red object, clean uncluttered composition, large
negative space, macro-leaning medium shot, deep blue-black color grading, the dark
red (#8A1F1F) strictly isolated, restrained archival aesthetic, cold and sharp
--ar 16:9 --v 8.1 --style raw --sref [锚定码] --sw 200
--no colorful, multiple red elements, red lighting everywhere, neon, cyberpunk,
saturated palette, text, watermark, logo
```

要点：专测"全画面近单色、只留一点红"。此张红若控不住（满屏泛红），后续所有异常值镜头都会废 —— **风险最高，最该先验证**。

## 验证清单

- [ ] 灰白人物 / 黑蓝底 / 暖黄字三色焊得住？
- [ ] 截帧④ 的红控住没有？
- [ ] 4 张是否像"同一个系统导出的档案"？
- [ ] 焊得住 → 锁参数进分镜；焊不住 → 只调 sref 码 / `--sw`，不动画面内容。

# 49 ·《群羊的低语：觉醒时刻》AI 蒙眼剪辑复刻实验

> **实验命题**：用 AI（Claude）+ 代码替代人工剪辑环节，从同事拆解笔记里的原始 AI 直出素材，
> 复刻出一部成片。对标基准 = 同事手工剪辑的「成片二稿 03:09」。
> **结论**：成立。全程无剪辑软件、AI 不"看"画面，靠分镜逻辑 + ffmpeg 代码装配，产出 02:51 成片。

## 怎么做出来的（关键链路）

1. **读源**：飞书拆解笔记（登录态经浏览器读取）→ 提取剧本/分镜/素材清单。
2. **下素材**：浏览器内 `fetch` 片段→POST 到本地接收器落盘（令牌不出浏览器，绕开下载守卫）。27 个片段时长全部与笔记登记吻合，校验通过。
3. **要字幕**：OCR 引擎在本机不可用 → 改"成片字幕带蒙太奇拼图 + 人工读图"得到**地面真值字幕脚本**（比分镜精简台词更完整）。
4. **装配**：`config.py` 定镜头顺序/字幕/人物卡，`build.py` 用 ffmpeg 统一规格→拼接→生成 ASS→烧字幕+混音。
5. **自检**：抽帧验证字幕字体/描边/居中无乱码、人物卡与角色匹配、音画在轨。

## 与基准的差异（诚实记录）

| 维度 | 成片二稿（人工） | 本片 v1（AI） |
|---|---|---|
| 时长 | 3:09 | 2:51（用更短补镜头版 + 未含分镜单元20-21蒙太奇收尾） |
| 镜头粒度 | 有明显裁剪/压缩（天台回溯被压到~20s） | v1 用整段镜头（更从容，留待反馈裁剪） |
| 字幕 | 主角误作"陆远/陆总" | 按角色资产校正为**杜远** |
| BGM | 外部添加 | 源笔记无 BGM 文件，留配置槽（创作者审美决策） |

## 待创作者反馈的点（蒙眼剪辑法 = 反馈驱动）

- **BGM**：放音乐到 `02_素材/audio_ref/` 并设 `config.BGM_FILE`，重跑即自动 -16dB 垫入。
- **节奏**：若要贴近人工版的快切，给镜头加裁剪点（config 可扩展 `trim`）。
- **字幕时机/人物卡**：改 `config.py` 后 `python build.py --final-only`（秒级出片，复用已编码片段）。

## 一键复现

```bash
cd 04_脚本
python build.py            # 全流程（首次，约5-8分钟）
python build.py --final-only   # 只改了字幕/人物卡时，秒级重出
python build.py --probe        # 只看时间轴不渲染
```
依赖：ffmpeg、Python3、系统中文字体（SimHei/微软雅黑）。

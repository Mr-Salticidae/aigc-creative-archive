# 剪辑流程与时间轴 ·《群羊的低语：觉醒时刻》第一幕

> 用「蒙眼剪辑法」让 AI 替代人工剪辑：AI 看不见画面，但靠分镜逻辑 + 代码精确装配；
> 创作者负责审美判断与精确反馈。本文 = 流程记录 + 可审核时间轴草案。

---

## 一、六步工作流落地

| 步骤 | 本项目执行 |
|---|---|
| 1 素材确认 | 从飞书拆解笔记下载 20 个直出镜头 + 7 个补镜头，全部 960×720 / 60fps / 含原生音频，时长与笔记登记一致（已校验，见 `01_立项`）。 |
| 2 节奏草案 | 笔记分镜本身按 10s/单元 切好，且素材命名 `场景-单元号` 与单元一一对应，"按编号顺序拼接"即成片 —— 草案即下方时间轴。 |
| 3 双轨映射 | 有角色台词。BGM=骨架（本片缺源，留槽）；台词=性格钩子（白雄威压/赵啸狂躁/杜远冷峻/铁山忠厚）；字幕=信息稳定输出。 |
| 4 代码实现 | `04_脚本/config.py`（配置即真相）+ `build.py`（ffmpeg 装配）。改卡点/字幕只改 config。 |
| 5 反馈迭代 | 字幕、人物卡、卡点、响度均参数化；创作者给"第几秒+原因+期望"即可定位修改。 |
| 6 占位→替换 | 本片素材已是成片级直出，直接装配；补镜头按规则替换问题镜头。 |

## 三、字幕脚本来源（关键）

台词以**成片二稿烧录字幕为准**（地面真值），通过 OCR 受阻后改用「字幕带蒙太奇拼图 + 人工读图」逐条转写（见 `02_素材/reference/_montage`），比分镜笔记的精简台词更完整。
- 修正：成片字幕将主角写作"陆远/陆总"，与剧本/角色资产「杜远(Du Yuan)」不符，判为笔误，本片统一用 **杜远**。
- 风格对标：白字 + 黑描边 + 底部居中、短句快显（漫剧惯例）。

## 四、人物登场介绍卡

成片二稿未见明显人物卡，本片按"加人物登场介绍"需求补做 lower-third 名牌（首次登场浮现，金色名+白色身份）：
杜远(多塞特羊·重生归来者) / 白雄(雄狮·万钧掌权者) / 赵啸(斑鬣狗·万钧二少) / 白素清(布偶猫·冷面执行人) / 铁山(黑犀牛·重装接应)。

## 五、音频与 BGM

- 直出片段自带原生音频（SFX/环境/配音），逐片响度不一（实测 -15~-33 dB），故最终 `dynaudnorm` 统一响度。
- 源笔记**无独立 BGM 文件**（同事在剪辑环节外部添加），BGM 属创作者审美决策：已在 `config.BGM_FILE` 留槽，放入音乐文件即自动按 -16dB 垫入混音。

## 六、如何迭代（反馈协议）

精确反馈 → 改 `config.py` → 重跑 `python build.py`：
- "某句字幕太早/太晚" → 调该 shot 的 `subs` 顺序或 `SUB_PAD`。
- "某镜头太长/太短" → 蒙眼剪辑 v1 用整段；如需裁剪，在 config 给该 shot 加 `trim`（可扩展）。
- "人物卡挡画面/时机不对" → 改 `INTRO_CARDS` 的 `at/t/dur`。
- "字幕太大/描边太重" → 改 `SUB_FONTSIZE/SUB_OUTLINE`。
- "要加 BGM" → 放文件并设 `BGM_FILE`。

# 《众》抽图作业单 v1（照着执行）

> 配合 `首帧MJprompt_v1.md` 用。MidJourney 出 6 张静态首帧 → 喂可灵。
> 三个小拍板项已按推荐锁定（执行中可改）：**片长 80s/6 拍**；**L2/L3 = 近实留媒介余烬**（不追全写实，躲撞车池+保一致）；**第5拍特写顺序 = 老球迷 → 孩子 → 独自来的人 → 球员（4 个）**。

---

## 0. 开抽前·MJ 设置（一次）
- 版本最新、`--style raw` 已在每条 prompt 里，无需再设。
- 关掉"自动加风格化"的默认高 stylize：我们靠 prompt 里的 `--stylize 200/220` 控。
- 准备一个地方贴 **锚 A / 锚 B 的图 URL**（拿 URL：网页版点开大图 → 右键复制图片地址；或 Web 端 … → Copy → Image Address）。

---

## 1. 批次顺序与卡点（务必按这个序，sref 要逐级传）

### 批次①：抽锚 A（皮影签名）——决定全片刻法，最重要
- 抽 **Frame 2「聚集」**，一次 4 格，**重抽到 6–8 组**(约 24–32 张)里挑。
- **选片标准（锚 A 必须同时满足，缺一重抽）**：
  1. 平面背光剪影感强，**不是 3D/塑料光**；
  2. 半透皮料能看出"透光"，镂刻花纹像花边透亮；
  3. 限色干净：暖白底 + 琥珀红/赭褐 + 描金,**没有跑出蓝绿紫杂色**；
  4. 人群是"一片集体生物"的密度,不是稀稀拉拉几个;
  5. 边缘是"切皮"的硬朗,不是水彩晕。
- 选定 1 张 → **Upscale → 复制图 URL = 锚 A**。
- ⛔ **卡点 GO/NO-GO**：锚 A 不够"皮影"就别往下走,全片签名都靠它。**抽完把你选的这张发我,我帮你判过不过关再继续。**

### 批次②：抽锚 B（近实·媒介余烬）
- 在 **Frame 4** 的 prompt 前加上 `--sref <锚A图URL> --sw 30`，抽 4–6 组。
- **选片标准**：现代看台是"近实"的活气,但**全画面浮一层半透琥珀颗粒 + 人物有背光描边 + 隐约戏布经纬**;不能变成普通体育摄影(那就丢了媒介,撞车池)。
- 选 1 张 Upscale → 复制 URL = 锚 B。
- ⛔ 卡点：B 帧若毫无皮影余烬味,把 `--sw` 从 30 提到 45 再抽。**拿不准也发我看。**

### 批次③：抽 L0 静镜（Frame 1）
- prompt 前加 `--sref <锚A> --sw 80`，抽 2–3 组。
- 选片标准:一个**平凡、不英雄、略孤独**的小人物 + 大留白;若 MJ 给得太华丽/太悲情,重抽(我们要"普通")。

### 批次④：抽命题镜（Frame 5）
- prompt 前加 `--sref <锚B> --sw 60`（这张是 L3 实感,靠 B）,抽 4–6 组。
- 选片标准:① 几个普通人特写有"各自是主角"的平等感;② **签杆能看见、且彼此相连、没有一个总操纵者**(这是命题机关,杆子没出来就重抽);③ 有**一处孔雀蓝冷点**从暖调里跳出来。
- 这张概念最满,**多抽,选最读得懂"互为提线"的那张。拿不准发我挑。**

### 批次⑤：抽两张渡变（Frame 3、Frame 6）——最难,放最后
- Frame 3 加 `--sref <锚A> --sw 60`;Frame 6 加 `--sref <锚A> --sw 70`。
- **各多抽 8–12 张**,专挑**"中间态"最自然的**:
  - Frame 3 要的是"前排已长出血肉、后排还是皮影"的同框过渡,口型不变;
  - Frame 6 要的是"血肉正被拍扁回皮影、最后凝成一只蹴鞠"的收束。
- 中间态可遇不可求,**把候选发我一起挑**,这两张决定古今咬不咬得上。

---

## 2. 选片总口诀（每张都过一遍）
- **像皮影吗?**(L0/L0′)/**还留皮影余烬吗?**(L2/L3)——丢了媒介就废。
- **限色干净吗?**——跑杂色重抽。
- **命题在吗?**——平凡感(F1)、所有人是主角+提线(F5)、咬合(F3/F6)。
- **可灵好动吗?**——构图给运动留了空间(留白/同向/可后退镜头),别选满到没法动的。

---

## 3. 回报点（你抽我审，三个 GO/NO-GO）
1. **锚 A 选定** → 发我审(全片签名)。
2. **锚 B + Frame 5** → 发我审(余烬味 + 提线机关)。
3. **Frame 3、6 候选** → 发我一起挑中间态。
其余(F1)你自己按标准定即可。

## 4. 抽完之后
6 张静态 → 我们排 **80s 静态占位粗剪**(每拍时长、字卡「一万年,一个声音」位置)→ 可灵逐级 L0→L1→L2→L3→L0′ 分段图生视频 → 后期叠戏布/灯影/限色/颗粒过渡 + Suno。

## 0. 决策链（为什么是「木刻 ×《众》」）

本片不是一次性挑出来的，是四轮收敛的结果，记录在案以便复盘：

1. **类型**：短片（可灵 3.0，≥20s，720p+，画幅不限）。
2. **赛制判断**：可灵官方征集 = 平台/评委选，**非同行互投**。知识库 [[同行互投赛制的反主流原则]] 有明文死边界："不适用评委/平台/公众"。→ 反主流触发器在此**不适用**。但本片定位为作者表达，**主动放弃获奖校准**，不为"燃 + 完成度"妥协深度。
3. **深度地基**：放弃奇观路线（声浪/人浪/光柱=美而不险，停在"佩服"），换叙事/命题地基（[[浪漫题的两条路线_奇观vs叙事]]：震撼≠深度）。
4. **表现形式破同质化**（决定性一步）：用**媒介**做差异化（[[红海主题的三条破局路径]] 路径三·媒介稀缺的最高级用法）。避开 AI 动画当前撞车池（见 `05_参考文档/国际AI动画影展_风格地图`），选**木刻版画**——因为**媒介本身就是主题**。

## 1. 核心方案

- **一句话故事**：无数球迷被木刻成同一块木板上的一个集体生物——它一起吸气、一起前倾、爆发出一声呐喊；镜头最终钻进这庞然大物，发现每一道刻痕里都是一个具体的、孤独的人。
- **核心命题**：个体如何在呐喊里溶进集体，又如何在集体里仍然是自己。应援＝人短暂地不再孤独。
- **主体**：无固定主角。一个"人海集体生物" + 钻进刻痕后浮现的若干具体个人（各出现一次）。
- **观看规则**（前 15s 建立）：黑底白刻线 → 字幕「一万个人，一个声音。」→ 镜头从一片刻满人头的"人海木板"极缓拉开，人海随一次进攻**整体前倾** → 观众立刻读懂"这是一个活的集体"。
- **主要一致性**：**风格一致性（木刻语言）+ 命题一致性**——不是角色一致性。这是 AI 可制作性的胜负手（每段不同人、只出现一次，零跨镜头人脸一致性负担）。
- **情绪曲线**：孤独的一张脸 → 溶进人海 → 集体爆发（全片唯一"热"刻）→ 钻回一道刻痕里的一个人 → 散场后空木板上残留的几道刻痕。**深度在涨，不是音量在涨。**
- **结尾余味**：人海散去，木板上只剩稀疏几道没擦净的刻痕（致敬 Kentridge 擦除美学）——集体走了，个体的痕迹留下。尾字卡留白，或「散场之后，我们各自是谁？」

## 2. 视觉语言系统（Look Bible）

- **限色**：黑 + 1 主色（队色＝深红）+ 纸白；偶尔第二专色（暖黄＝球场灯）。
- **刻痕即运动**：人海的"动"用**刻线方向的流动**（平行斜线扫过＝声浪）表现，而非真实位移 → 规避 AI 大幅运动短板。
- **留白即呼吸**：版画纸白负空间承担"安静"。
- **颗粒**：叠真实木纹 / 油墨网点 → 强化"手刻"非数字感。
- **人脸＝木刻面具**：粗刻线五官，直接消灭恐怖谷（不再依赖逆光躲脸）。

---

## 3. 关键镜头（5 个，均木刻语言）

1. 单人脸特写，粗刻线，一双眼睛 → 镜头拉开，这张脸**只是人海木板上的一个头**。
2. 人海整体随进攻前倾，斜刻线如声浪横扫全画。
3. 爆发：黑红两色对冲，纸白炸裂，围巾木刻化作飞鸟群升起。
4. 镜头钻进刻痕，人海重新解析成一个个孤独的具体的人（老人、孩子、独自来的人）。
5. 散场空看台，空木板，残留几道刻痕，油墨未干。

---

## 4. AI 制作路径（可行性已查证）

- **首帧**：MJ/SD 出木刻图——关键词族：`linocut / woodcut print / German Expressionism / Lu Xun woodblock / bold carved black lines / limited palette (black, deep red, cream) / halftone ink texture / high contrast`。**sref 锁版画作者签名**（全片统一刻法，遵 [[sref编号独立律]]）。
- **运动**：可灵 3.0 图生视频。遵其铁律——**中文 prompt、全正向句（无负向框）、一镜一主导动作、慢速、景别单选**。木刻平涂运动幅度小，最适配可灵。
- **后期**：叠木纹/网点颗粒、限色调色，把"数字味"压成"印刷味"；声浪床 + 字卡（蒙眼剪辑法 config）。

## 5. 待决问题（立项遗留，逐项拍板）

1. **风格定稿**：木刻（推荐）/ Risograph 丝网印 / 油画玻璃（Petrov-Miailhe）三选一。
2. **片长**：80s（5 段）还是拉到 ~120s（加"钻进刻痕"的多个个体）。
3. **队色**：深红（推荐，木刻黑红经典）/ 其他；是否引入第二专色暖黄。
4. **结尾字卡**：留白 / 「散场之后，我们各自是谁？」二选一。
5. **是否保留"围巾"母题**：木刻版里围巾化作飞鸟群（第 3 镜）是否保留，还是纯人海。
6. **配乐**：是否叠 Suno（声浪床之下极简弦乐，仅爆发段进入）。

## 一句话

无数球迷被**木刻**成同一块木板上的一个集体生物——它一起吸气、一起前倾、爆发出一声呐喊；镜头最终钻进这庞然大物，发现每一道刻痕里都是一个具体的、孤独的人。

## 核心命题

个体如何在呐喊里溶进集体，又如何在集体里**仍然是自己**。应援，是人短暂地不再孤独的方式。

## 关键决策（为什么是这个形态）

1. **赛制判断**：平台官方征集 ≈ 评委选，**不是同行互投** → 反主流打法不适用；但本片为作者表达，不追获奖校准。
2. **表现形式破同质化**：放弃写实，做动画；主动避开当前 AI 动画三大撞车池（羊毛毡 Felt Realism / 中国水墨 / 油画散文片）与两大引力井（新海诚渐变天、皮克斯光滑 3D）。
3. **媒介即主题**：选**木刻版画 / Linocut**——版画自诞生即是"群体/劳动者/集体发声"的媒介（德表现主义、墨西哥 TGP、鲁迅新兴木刻、反种隔 APART），与"球场应援=工人阶级集体仪式"严丝合缝。

# 47_众（The Multitude）· 球场应援大赛动画短片

> 立项 2026-06-23 ｜ v3 结构升级 2026-06-23 ｜ 状态：⏳ 概念锁定 v3，待出首帧
> 投稿：可灵「球场应援大赛」（投稿 2026-06-12～07-26，揭晓 08-26）
> 性质：作者短片。比赛只是发表由头，**获奖不是目标，进作品集才是**。
> 形态：**皮影⇄现代「媒介随古今融化」· 六拍首尾呼应**（v3）。v1 木刻方案已存档。

## 一句话
一个古代平凡、孤独的小人物被皮影刻在戏布上；他和无数同样平凡的人聚成人海，一声呐喊里皮影熔成血肉、戏布变成今天的世界杯现场；镜头挨个特写普通观众与球员，发现**所有人都是主角**，头顶重新浮起彼此相连的签杆；终场胜利的呐喊中血肉又熔回皮影，凝成一只古老的蹴鞠——一万年，换了皮囊，没换那一声。

## 完整节奏（6 拍）
1. 古代·平凡孤独的人（纯皮影 L0）→ 2. 聚集（L0）→ 3. 呐喊·渡变①皮影→现代（L1）→ 4. 足球现场·近实带媒介余烬（L2）→ 5. 观众/球员特写·所有人都是主角·提线重现（L3）→ 6. 胜利呐喊·渡变②现代→皮影蹴鞠（L0′，首尾呼应）。

## 核心命题
个体如何溶进集体、又在集体里**仍然是自己**。
- **暖落法**：人海里每个普通人，特写过去都是主角。
- **深机关**：互为提线——人海挑动球员、球员挑动人海，无总操纵者，谁都是主角谁也不是傀儡（焊在第 5 拍）。

## 关键决策
1. **媒介即主题·终极形态**：蹴鞠=足球起源、皮影=活动影像起源；且**媒介自己的生死（皮影→血肉→皮影）就是古今那条时间线**。
2. **媒介熔解五级阶梯**（控风险核心）：L0 纯皮影 → L1 半熔 → L2 近实带余烬 → L3 实·提线重现 → L0′ 熔回。融化＝排台阶，连续变形只在第 3、6 拍两镜。
3. **避同质化**：皮影不在 AI 动画撞车池/引力井内；L2/L3 留"媒介余烬"躲写实撞车池。
4. **⚠️ 最高纪律**：传统文化与媒介融化都是燃料，内核必须是命题——死守不滑向介绍片。

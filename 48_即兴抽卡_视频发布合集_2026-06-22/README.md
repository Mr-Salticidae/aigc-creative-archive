> **07/08/09 封标说明（主题不同，按题材各自定制，脚本 `_build_covers_themed.py`）**
> 前六组走统一「古风模板」（宋体竖排 + 楷体诗句 + 暖金细线，`_build_covers.py`）。07 起主题跳出古风，沿用同模板会失味，故按各自气质重做封标（依据 skill `aigc-poster-layout` 的"封标设计语言随题材切换"），并参考网络成功案例：
> - **07 假面**（暗调戏剧）：骨白宋体竖排主标 + **朱红**细竖线 + 「戲」字朱砂印章 + 角落柔暗（护脸/护裂面），以血红 + 大留白制造戏剧张力。
> - **08 栖白**（时尚编辑）：顶部 **Playfair Display（Didone 高反差衬线）** 报头「BLANC」+ 细横线 + 微软雅黑封面语（精简两行）左下 + 宋体作品名「栖白」右下，仿时尚杂志封面。字体随项目存于 `_fonts/PlayfairDisplay.ttf`（OFL）。
> - **09 霜瞳**（高调薄雾·冷调）：原图高调薄雾、近纯白底，须用深字才读得清——冷墨蓝宋体竖排主标「霜瞳」落进左上薄雾留白 + **冰蓝**细竖线（呼应蓝瞳与刃光）+ **白色柔光晕**托字保可读，空灵冷调贴合仙侠 cos。
> - **10 蝶隐**（梦核薄纱·珍珠留白·编辑式）：初版葡萄紫宋体过闷、缺高级感，已重构——改走**编辑式留白**：**珍珠白仿宋**竖排主标「蝶隐」（细劲、大字距/行距）落在左上**暖 aubergine 柔暗角**里托出（不破梦核高调）+ **香槟玫瑰金**细竖线与短线 + **Playfair** 小号拉丁落款「PSYCHE」（蝶＝灵魂）。低饱和珍珠+香槟取代高饱和紫，呼应暖肤光，空灵而高级。脚本中以 `python _build_covers_themed.py 10_` 单独重建本组、不动旧组。

## 发布建议（依据 content-publish-sop）

1. **入场票审计**：九组均属"美学辨识度"型，无国民级文化预载，渗透率中等。结论——**先同步发布、不投流，等自然流量验证**；完播率 >30% 且互动率 >3% 再考虑投流放大。
2. **主推顺序**：`03 黑蔷薇`（彩窗废墟+黑玫瑰对比）、`07 假面`（红衣裂面，张力最强）、`05 噤声`（食指抵唇手势+红梅）、`06 孤鸿`（古风男像+荒野抚琴，男性题材稀缺）前 3 秒识别度最高，建议首发测水温；`09 霜瞳`走二次元/cos 垂类流量池单独试水；`10 蝶隐`走西幻/精灵/梦核（fairycore）垂类流量池单独试水；`08 栖白`等偏收藏向，其余组同步发布观察数据。
3. **封面**：竖屏平台（快手/抖音）用 `3:4`，B站横屏用 `16:9`，`4:3` 备用。封面已做模糊铺底 + 主体居中 + 标题，能独立传达"这是什么"。
4. **标题/标签**：各组 `简介_第一人称散文诗.md` 内已按平台给出标题与 2–3 个精准标签，优先内容类型标签（#AI绘画 / #AIGC），不堆砌。
5. **合规**：发布时如实勾选"AI 生成 / AIGC"标注。

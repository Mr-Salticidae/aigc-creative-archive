## 工作流（一期一角的标准节拍）

1. 选定角色 → 读对应角色卡（Layer 1 身份锁 + 写实化补充）
2. 按 `视觉风格基底_写实.md` 出**写实原图**（默认 MJ；含中文字的合成图用 GPT Image 2）
3. 按 `素材包结构规范.md` 出**封面海报 + 轮播图**
4. 套 `文案模板.md` 写**小红书文案**（沉浸独白口吻 + AI 标注）
5. 归档进 `EPxx_角色名/`，复盘记录进知识库

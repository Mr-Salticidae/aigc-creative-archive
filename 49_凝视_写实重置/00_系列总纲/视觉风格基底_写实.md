# 视觉风格基底 · 写实

> 本文件是写实重置的**唯一视觉标准**，取代旧版「日漫厚涂 + niji 7」规范。
> 遵循 prompt 工程铁律：极简化、三段式、风格基底前置、词系互斥、负面词疫苗。

---

## 一、风格基底（前置，所有角色共用）

> 三段式：**[风格基底] + [角色 Layer1+Layer2] + [构图动作 Layer3]**
> 基底永远写在最前面，锁住"这是一张电影感真人写真"。

**基底句（中英对照，实际出图用英文）：**

```
cinematic portrait photography, photorealistic, shot on 85mm,
shallow depth of field, low-key chiaroscuro lighting,
desaturated muted film tones, fine film grain, real skin texture
```

中文备忘：电影感人像摄影 / 真实照片质感 / 85mm 定焦 / 浅景深 / 低调明暗对照光 / 低饱和胶片色调 / 细腻胶片颗粒 / 真实皮肤质感。

## 二、默认工具与参数

| 用途 | 工具 | 参数 |
|---|---|---|
| 写实原图（无字） | **Midjourney**（默认） | `--ar 3:4`（封面/轮播） 或 `--ar 9:16`（沉浸大图）`--style raw --stylize 150` |
| 含中文字的合成海报 | **GPT Image 2** | 中文成字更好，封面海报标题/独白排版交给它 |
| 角色一致性 | MJ `--sref <编号>` 锁画风 + `--cref` 锁脸 | sref 编号撞墙先换号（见 [[aigc-character-consistency]]） |

> 为什么海报用 GPT Image 2：01 海报（SCARLATTA／斯卡拉塔 + 中文独白）即由它合成，中文字形完整。写实**人像本体**仍走 MJ，文字排版层叠加。

## 三、负面词疫苗（写实必打，防破真人错觉）

```
--no illustration, anime, cartoon, 3d render, cgi, plastic skin,
smooth airbrushed skin, oversaturated, lowres, deformed hands,
extra fingers, watermark, text artifacts
```

重点防：插画感 / 塑料皮肤 / 磨皮过度 / 过饱和 / 崩手 / 多指 / 水印。

## 四、互斥词系（别混用，会互相打架）

| 想要的方向 | 用 | 别同时写（互斥） |
|---|---|---|
| 真实照片 | photorealistic, film grain | painterly, illustration, niji |
| 暗调 | low-key, chiaroscuro | high-key, bright, cheerful |
| 浅景深焦点 | shallow DOF, 85mm | wide-angle landscape, deep focus |

## 六、一致性策略：**回避，而非死磕**（系列铁律）

> 经验结论：**MJ 美学最高，但跨图人物一致性差。** 与其和它死磕脸的一致性，不如绕开。

- **每期只保留一张"定妆正脸"**（= 该角色的写实原图），作为轮播本体大图。
- **其余所有图一律 MJ 细节/氛围/局部镜头，不出现角色正脸**：道具特写、手部、织物材质、背影/侧影、半遮局部（只露一只眼）、环境氛围。
- 这些镜头**不需要脸的一致性**，MJ 的不稳定就伤不到——短板绕开，美学长板吃满。
- **靠"指纹元素"维持识别**，而非靠脸：每角色有专属道具/主色（SCARLATTA = 红丝绒 + 银裂面具 + 红甲），细节图反复出现这些"指纹"，观众照样认得出是谁。
- 实在要多出一张人物图：用同一张原图做**裁切/局部/调色变体**，不重新 roll 脸。

> 推论：封面海报、本体大图用那张定妆原图；轮播靠 MJ 堆"高级细节"。详见各期 `轮播图_出图指令.md`。
> 参考 [[aigc-character-consistency]]——本项目**主动选择不用**四层金字塔，因为写实细节流不依赖锁脸。

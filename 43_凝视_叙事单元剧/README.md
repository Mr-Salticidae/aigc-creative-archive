## 项目定位

把《凝视 The Gaze》IP 从"角色肖像系列"延展为**长线更新的叙事单元剧系列**,从 VERITIA(虚君)入手。

三条创作铁律(立项时定下,不可滑坡):

1. **不做介绍类视频**——介绍类容易滑向自我感动。每集必须是故事。
2. **代入感优先**——每集 = 一个戏剧问题 + 一次选择 + 一个代价。观众代入的不是"她好美好惨",是"换我,我怎么办"。
3. **迷雾点亮模式**——每集是世界观故事迷雾的一个小版图被照亮。前提:迷雾先在创作者手里全亮(世界观圣经先行,参照 FromSoftware 碎片化叙事)。

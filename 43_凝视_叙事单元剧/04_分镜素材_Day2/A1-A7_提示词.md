## A1 · 王座厅广角(镜头 02,3.0-9.0s)

**叙事**:开场静默铺垫,确立"她坐在末代王朝王座上"的世界观。

**简化提示词**:

```
中远景广角,镜头从下方仰望末代王朝衰败王座厅。
画面中央是一把沉重华丽但已破败的金红色王座,
她端坐其上,身形相对较小但庄重,玫瑰金长发垂落,
身着暗红丝绒长袍,头戴破碎金王冠。
背景是高耸但部分崩塌的拱柱,残余彩窗光从远处洒入。
画面笼罩在 chiaroscuro 戏剧光影中,deep amber + dark red + cold shadows。
9:16 竖构图。注意画风和面部细节的一致性。
```

## A5 · 镜中王朝崩塌(镜头 08,65.0-85.0s)

**简化提示词**:

```
复合构图。前景半侧:她侧身站在画面右侧,玫瑰金长发垂落,
身着暗红丝绒袍,头戴歪斜破碎金王冠,
她的目光投向画面左侧的一面巨大破镜。
镜中映出的不是她,而是远处王朝崩塌的画面 ——
火光、烟雾、坍塌的拱柱、流离的百姓剪影 ——
镜面有几道裂痕从中心辐射,真相透过裂痕可见。
chiaroscuro 戏剧光影,前景偏暗内省,镜中映像偏明带火光。
9:16 竖构图。注意画风和面部细节的一致性。
```

**特殊建议**:这张可以试用 V-02(王冠破碎那张)作为画布输入 1 替代主图,因为这张主体不在面部,画风延续比角色 ID 锁定更重要。

末代王朝的颓败与寂静。
9:16 竖构图,画面中无人物。注意画风和面部细节的一致性。

## 验收标准(每张通用)

- ✓ 9:16 竖构图,≥1024×1820
- ✓ 画风一致性:painterly + chiaroscuro,与 Day 1 主图同质感
- ✓ 角色一致性:玫瑰金长发 / 天蓝眼 / 破碎金王冠 / 暗红丝绒袍 全部到位
- ❌ 不出现:笑容 / 愤怒 / 明亮场景 / 多角色同框

通常 2-3 轮可拿到满意成品。出图后丢回来,我对照剧本叙事 + Day 1 风格做评估。

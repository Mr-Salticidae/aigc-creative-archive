## 验收标准

每张图至少需要满足:
- 9:16 竖构图,≥1024×1080(后续会缩放到 1080×1920)
- 角色一致性:玫瑰金发 / 天蓝眼 / 破碎金王冠 / 暗红丝绒袍 全部到位
- 画风一致性:painterly + chiaroscuro,与 Day 1 主图同质感
- 不出现:笑容 / 愤怒 / 多角色同框 / 明亮场景

通常 2-3 轮可以拿到满意成品,5 轮以上还没到位说明 prompt 需要调整。

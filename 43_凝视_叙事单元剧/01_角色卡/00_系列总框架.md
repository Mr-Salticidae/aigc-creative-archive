## 6 种凝视的本体论

The Gaze 系列展示的不是"6 个角色",而是 **6 种凝视方式的具象化**——每个角色都是一种凝视的化身。

| 编号 | 角色 | 凝视类型 | 关键词 |
|---|---|---|---|
| 01 | SCARLATTA | 审判性凝视(JUDGMENTAL GAZE) | 看穿伪装 |
| 02 | VESPER | 暧昧性凝视(LIMINAL GAZE) | 看见两面 |
| 03 | NULL-07 | 技术性凝视(DATA GAZE) | 看见数据 |
| 04 | MORGANA | 预知性凝视(PROPHETIC GAZE) | 看见未来 |
| 05 | SELENWË | 守护性凝视(VIGIL GAZE) | 看见入侵 |
| 06 | VERITIA | 真相性凝视(TRUTH-PIERCING GAZE) | 看穿幻象 |

---

## 层级结构

```
最外层(物理)            ← SCARLATTA
   ↓ 看穿对象的伪装

中介层(现实与虚拟之间)  ← VESPER + NULL-07
   ↓ 看穿"昼夜之间"和"数据之间"的边界

时间层(超越当下)        ← MORGANA + SELENWË
   ↓ 看见尚未发生 / 看见已经发生

最内层(本体)            ← VERITIA
   ↓ 看穿凝视这件事本身
```

**VERITIA 是终点**:她质疑"凝视"本身——即使是凝视也可能被中介。但她仍选择相信自己直接的看。

## 跨角色一致性原则

### 整套系列的视觉统一标识

无论哪个角色,所有图片必须保持:

- 半写实日漫风格 (semi-realistic anime illustration style)
- Dutch angle 倾斜构图 (tilted dutch angle composition)
- 暗调氛围 (dark moody atmosphere / chiaroscuro lighting)
- 大特写镜头 (extreme close-up framing)
- 前景遮挡元素 (foreground occlusion - mask/glass/HUD/bow/etc.)
- 一只露出的眼睛是视觉焦点 (one revealed eye as focal point)
- painterly 厚涂笔触 (painterly rendering)
- 戏剧性 chiaroscuro 光影
- 9:16 竖版构图(系列首选)

### 通用 prompt 前缀模板

```
[角色独有特征,见各角色卡 Layer 1],
semi-realistic anime illustration style, dramatic chiaroscuro 
lighting, deep shadows with [角色色调,见 Layer 2] accents, 
dust particles floating in light beams, glossy metallic 
highlights, hyper-detailed eyes with melancholy depth, 
painterly rendering, cinematic shallow depth of field, 
tilted dutch angle composition, [角色专属氛围词,见 Layer 2]
--ar 9:16 --niji 7
```

### 系列禁忌(避免破坏 IP 一致性)

- ❌ 写实风格 / 卡通风格 / 像素风格
- ❌ 明亮欢快的氛围
- ❌ 全身像或风景为主的构图
- ❌ 让角色微笑或大笑
- ❌ 添加未在角色卡中定义的新元素
- ❌ 使用 GPT Image 2 等"清水插画感"工具(画风差距大,会破坏系列一致性,除非你能用 prompt 强制对齐)

## Prompt 三层结构(所有角色卡通用)

每个角色的 prompt 都按以下三层组织,便于复用和迭代:

### Layer 1:角色 ID(必须保留,锁住"她是谁")
角色独有的身份特征、关键道具、独有元素

### Layer 2:画风技法(必须保留,锁住"她长什么样")
所有角色共享但各有色彩偏好的画风词汇

### Layer 3:构图动作(可以替换,这是变量)
镜头景别、姿态、场景、特定瞬间——这是 V-01/V-02/V-03 等不同分镜的差异点

## 角色卡使用指南

### 用于 AI 出图工具

1. 打开对应角色卡
2. 复制"视觉特征 / Layer 1+2"段落
3. 粘贴到工具(MidJourney / Niji / 可灵 Omni / 其他)
4. 按需要追加 Layer 3 的构图变化
5. 出图后对照"关键元素清单"逐项检查

### 用于 AI 视频工具(Kling / Runway 等)

1. 用对应角色卡 + Day 1 主图作为参考输入
2. "核心特征"字段填写角色卡的 Layer 1+2 浓缩版(<100 字)
3. 视频 prompt 描述具体动作/镜头运动

### 用于剧本/独白创作

1. 阅读"世界观与性格"段落
2. 参考"独白"作为该角色的语言风格基准
3. 任何衍生文本必须**符合性格描述**,不要让她做角色卡禁忌的事

### 用于二创/衍生开发

1. 阅读"风格延展边界"
2. ✅ 部分可以自由发挥
3. ❌ 部分是 IP 底线,不能突破

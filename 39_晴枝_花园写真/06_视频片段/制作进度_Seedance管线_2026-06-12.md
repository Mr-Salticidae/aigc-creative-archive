- **四条运动提示词**(在脚本 `SHOTS` 里,按图定制):

  |---|---|---|
  | 01 亮调微笑直视 | 微风拂发、眨眼、笑意加深 | 缓慢推近 |
  | 02 垂眸仰光 | 叶影闪烁、抬眼半分又垂下 | 向右缓移+前景视差 |

  共同骨架:首帧锁定 + 0-2s/2-5s 时间段落 + 身份保持(`preserve her exact facial features`)+ 负面约束(no text/subtitles/watermark)。

## API 规格(已验证)

- 端点:`https://ark.cn-beijing.volces.com/api/v3/contents/generations/tasks`
- 模型:`doubao-seedance-2-0-260128`
- 正式跑参数:1080p / 5s / `ratio: adaptive`(跟随 3:4 竖版)/ `generate_audio: false` / `watermark: false`
- 首帧传法:`content[]` 内 `type: image_url` + `role: "first_frame"`,base64 data URL

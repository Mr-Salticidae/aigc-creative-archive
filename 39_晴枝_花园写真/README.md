# 39_晴枝_花园写真

## 项目定位

- 标题：《晴枝》
- 类型：AIGC 人像写真系列
- 阶段：角色卡建立 + Midjourney 首轮主视觉测试
- 建档日期：2026-06-07
- 核心方向：春末初夏、花园漫步、元气感、气质、轻故事感

## 创作判断

前期作品已经做过较多悲伤、沉郁、低气压氛围。本项目转向春末初夏的明亮生命力，但不做普通甜美花园少女。作品重点不是“更美”，而是让人物拥有稳定气质和轻故事感。

晴枝的故事感来自恢复、好奇和松弛：她像刚刚从漫长的室内生活里走出来，在花园、风、阳光和花影里重新感到生活可爱。

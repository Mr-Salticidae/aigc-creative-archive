# R2 moodboard 面孔锁定成功记录

## 输入图

- 成功图：`R2_favorite_晴枝_moodboard_face.png`
- 方法：使用 v0.1 面部特写 prompt，并加入 moodboard

## 关键结论

这次问题不在 prompt 细节，而在忘记启用账号级审美锚点。

R1 用纯 prompt 追“惊艳面孔”，可以得到好看的春日写真，但五官比例、骨相记忆点和账号 IP 底色不足。R2 加入 moodboard 后，面孔立刻更接近账号审美：脸部更和谐，眼神更有吸引力，发丝和叶影更有质感，整体更像“跳蛛先生会发的图”。

## 这张图赢在哪里

- 面部更惊艳：五官比例比 R1 更集中、更抓人。
- 发丝更好：凌乱但有美感，增加了真实摄影的瞬间感。
- 叶影更有戏：光影直接参与脸部塑造，不只是环境装饰。
- 情绪更准：有轻微笑意，但没有变成普通甜妹。
- 晴枝成立：比檐下更暖、更春天，但仍能看出账号审美底色。

## 方法修正

后续晴枝出图默认走四层金字塔：

1. personalize / moodboard：锁账号审美底盘。
2. 角色卡 / 文字锁定段：锁晴枝的气质与面孔方向。
3. R2 成功图作为候选 oref：锁脸部一致性。
4. 单图 prompt 与参数：锁具体镜头、季节、动作和画幅。

## 下一步建议

不要立刻改大 prompt。先围绕这张 R2 成功图做两件事：

1. 用同一 prompt + 同一 moodboard 再跑 4-8 张，确认这不是偶然命中。
2. 如果同类质量稳定，就把 R2 作为 oref 候选，开始半身花园图和行走图扩展。

## 注意事项

R2 可以作为 oref 候选，不建议作为 sref。sref 负责作者风格签名，直接用人脸成功图容易让主体形态污染风格锚点。

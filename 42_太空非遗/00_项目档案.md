## 创作路径

1. 发散阶段:把"非遗"拆成 **题材 × 视觉风格 × 叙事角度** 三维矩阵
   - 题材四类:天生出片型(打铁花/醒狮/鱼灯)、质感微距型(缂丝/青瓷/大漆)、人物叙事型(传承人的手/戏台后台)
   - 风格六路:纪实电影感 / 新中式时尚大片 / 国潮插画 / 超现实概念 / 赛博融合 / 微距纯质感
   - 叙事四角:正在消失的瞬间 / 非遗活在当下 / 工艺的一秒钟 / 器物即山水
2. 首轮选定"打铁花 × 纪实电影感",因**预判撞车**改道复古科幻
3. 复古科幻出两版:70s Sci-Fi 海报感(主力) + Cassette Futurism(变体),最终 70s 版成图
4. 另备"东方太空歌剧"叙事版(祠堂锻火),未跑,prompt 存档于下

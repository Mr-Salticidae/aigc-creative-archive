## 说明

- **封面是代码排版的杂志式版式**(亮暖纸底 + 大留白 + 细装裱框 + 宋体主标/楷体副标 + 拉丁 kicker),把版画原作当"框中印刷品",非直接上原图。延续无品牌标识偏好。
- **核心机制**:`sref 管皮,散文管心,风格词管样子`——三者正交。换题材只换那段情绪独白,版画质感不变。

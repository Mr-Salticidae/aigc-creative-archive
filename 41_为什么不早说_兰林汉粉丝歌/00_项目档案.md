# 《为什么不早说》项目档案 v1.0

## 一句话命题

把兰林汉“嘴硬”直播语录做成一首可传播的粉丝二创梗歌，用重复 hook 和直播辩解语气形成记忆点。

## 核心判断

本项目最终成立的关键不是“把语录塞满”，而是找到一句可以反复唱、反复接、反复传播的主钩子：

```text
早说啊
不早说呢
为什么不早说
```

围绕这个 hook，歌词把游戏失误嘴硬、逆风甩锅、战术解释、生活日常嘴硬压缩成一个统一人格：理直气壮地不认输。

## 曲风迭代

### 放弃路线

| 路线 | 问题 |
|---|---|
| 中文说唱 / boombap | 容易显得技术外壳大于梗本身 |
| 欧美现代 rap | 制作语言高级，但中文口语梗贴合度弱 |
| 轻快流行梗歌 | 好听但嘴硬感不足 |
| 祝福式荒诞梗歌 | 有反差，但太轻，不够顶嘴 |

### 定稿方向

```text
嘴硬宣言式流行梗歌
```

重点是：

- `早说啊` 要像嘴硬宣判，不像开心口号。
- 主歌保留直播口语感，不做过度文学化。
- 副歌保持重复和弹幕可接唱。
- Bridge 从游戏嘴硬扩展到生活嘴硬，增强粉丝记忆。

## 歌词结构

| 段落 | 功能 |
|---|---|
| Spoken Intro | 建立口头禅 |
| Verse 1 | 游戏失误嘴硬 |
| Pre-Chorus | 战术解释与强行合理化 |
| Chorus | 主 hook 固化 |
| Verse 2 | 逆风甩锅与版本解释 |
| Bridge | 生活日常嘴硬 |
| Breakdown | 主播与观众 call-and-response |
| Final Chorus | 传播段落 |

## 封面定稿

封面经历了真实截图、原始视频帧、2D 插画、3D 动画风、半写实二次元等方向。

最终选择：

```text
半写实二次元暗调
红黑银主色
直播嘴硬状态
音乐封面构图
```

选择理由：

- 真人照片路线容易在“像不像”和素材边界上摇摆。
- 过度 3D 动画化会变可爱，削弱歌曲的嘴硬气质。
- 半写实二次元可以保留压迫感、音乐封面感和直播辩解动作，同时不被真人相似度完全绑死。

- `04_正式封面/FINAL_cover_4x3_为什么不早说.png`

## 可复用经验

1. 粉丝梗歌的主 hook 比语料覆盖率更重要。
2. 直播语录适合 call-and-response，因为它天然有主播和弹幕的关系。
3. 中文口语梗不一定适合说唱，很多时候更适合口语旋律化。
4. “轻快”会削弱嘴硬人格，本项目需要偏重、偏硬、偏顶嘴。
5. 封面如果追真人像，容易被不像问题拖住；符号化风格更适合粉丝二创传播。

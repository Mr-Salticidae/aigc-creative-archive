## 创作定位

本曲不是常规中文说唱，也不是轻快梗歌，而是“嘴硬宣言式流行梗歌”。

核心 hook：

```text
早说啊
不早说呢
为什么不早说
```

封面定稿路线：

```text
半写实二次元暗调 / 红黑银 / 直播嘴硬状态 / 音乐封面
```

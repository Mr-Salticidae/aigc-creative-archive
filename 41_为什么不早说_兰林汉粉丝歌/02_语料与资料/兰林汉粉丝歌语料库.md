# 兰林汉粉丝歌语料库

更新时间：2026-06-10

用途：给“兰林汉嘴硬主题粉丝二创歌”做素材底稿。当前版本主要来自公开搜索结果和 B 站公开视频标题，适合先写歌的主题、钩子和梗框架；逐字经典语录仍建议从原视频或直播回放中人工补录。

创作策略更新：这首歌应当“原话优先”。粉丝传播力主要来自熟悉感，歌词里要尽量保留兰林汉原话、标题中被粉丝反复引用的短句、弹幕固定梗；二创改写只用于补足押韵、转场和副歌结构。

最新核心判断：真正的主 hook 不应是泛化的“嘴硬王”，而应是兰林汉式甩锅闭环：

```text
早说啊
不早说呢
为什么不早说
```

这组三连天然有节奏、能递进、适合观众跟唱，也能把“嘴硬、甩锅、死不认错”全部收进去。

## 3. 核心原话池

### A. 主 Hook：早说三连

优先级最高，建议作为副歌核心。

```text
早说啊
不早说呢
为什么不早说
```

用法：

- `早说啊`：短促起句，适合切分节奏。
- `不早说呢`：带一点委屈和甩锅，适合旋律上扬。
- `为什么不早说`：完整爆点，适合副歌落点或全场合唱。

推荐副歌骨架：

```text
早说啊，早说啊
你为什么不早说
不早说呢，不早说呢
这把怎么能怪我
```

### B. 收场 / 认输但不认错

适合做每段结尾，形成“下把循环”。

```text
快拉倒吧
没招了
蹩叭叭了
盐尽了
下把吧
```

用法：

- `快拉倒吧`：打断弹幕或队友，适合 spoken tag。
- `没招了`：承认局势，但不承认自己错。
- `蹩叭叭了`：强烈口语特色，适合保留原声感。
- `盐尽了`：建议作为谐音/粉丝梗保留，若要更通俗可在字幕里标注。
- `下把吧`：天然段落结束句，适合作为 Outro。

推荐段落尾句：

```text
快拉倒吧，没招了
蹩叭叭了，下把吧
```

### C. 被单杀第一反应

适合第一段 Verse，场景最明确。

```text
我没按出来
没按到
没按出血池
这波纯属意外
正常失误
他这是偷袭
阴我
正面打他打不过我
卡了
网络波动
键盘问题
我手滑了
点错了
```

建议用法：不要全部堆上去，挑 4-6 个做连续快嘴。

```text
我没按出来，没按到
没按出血池，正常失误
他这是偷袭，阴我
正面打他打不过我
```

### D. 逆风崩盘硬洗

适合第二段 Verse，从个人失误扩大到队友、英雄、版本、机制。

```text
不是我菜，是 XX 问题
我这英雄本来就弱，换个英雄早赢了
队友太坑，带不动
打野不帮我，没法玩
对面英雄太超标，根本没法打
我爆杀的人正在爆杀我的队友
这游戏机制有问题，没办法
```

高价值句：

```text
我爆杀的人正在爆杀我的队友
```

这句非常适合进歌词，因为它有强逻辑反转和强画面感。

### E. 强行解释 / 死不认错

适合 Pre-Chorus，把情绪推向“早说三连”。

```text
我其实是故意的，试探一下
你看不懂，这波是战略撤退
我这是在演，给对面机会
你们懂个屁，这是高端理解
绝不可能有事儿
我怎么可能输，只是没认真玩
```

推荐组合：

```text
你看不懂，这是战略撤退
我这是在演，给对面机会
你们懂个屁，这是高端理解
早说啊，为什么不早说
```

### F. 生活 / 直播日常嘴硬

适合 Bridge，制造从游戏嘴硬到生活嘴硬的扩展。

```text
我沈阳车神，开车不可能出事
是别人撞我
我一点不胖，只是壮
我没嘴硬，我只是实话实说
我这操作没问题，是你们眼光不行
```

推荐桥段：

```text
我沈阳车神，不可能出事
剐蹭了也不是我的错
我一点不胖，我只是壮
我没嘴硬，我实话实说
```

## 5. 旧版泛化语料分类

### A. 嘴硬否认型

功能：适合做副歌反复。

- 我没急。
- 我真没急。
- 这不是嘴硬，这是判断。
- 你只需要说，我硬不硬。
- 弹幕懂什么，这叫思路。
- 不是我菜，是这波不该这样打。
- 这局问题不大，下一波直接拿下。
- 别说废话，看我操作。

### B. 强行解释型

功能：适合 Verse，制造“越解释越像破防”的喜剧效果。

- 这波不是送，是战术伏笔。
- 我这个位置没有问题。
- 对面这个人有点东西，但也就那样。
- 我不是被单杀，我是在试探。
- 你们只看结果，不看过程。
- 这个游戏理解，不是一般人能看懂的。
- 我刚刚是故意给他机会。
- 别急，真正的节奏还没来。

### C. 破防边缘型

功能：适合 Pre-Chorus，让情绪往高潮推。

- 怎么都在 FK 啊。
- 谁又在带节奏。
- 别搞我心态。
- 你们这弹幕太离谱了。
- 我跟你讲，这把真不是我的问题。
- 兄弟们，不要乱说。
- 这也能怪我啊。
- 我真的服了。

### D. 猴哥身份型

功能：适合主歌开头或 Bridge，建立人物标签。

- 国服第一猴。
- 世界最强猴王。
- 猴哥上线。
- 火眼金睛看穿峡谷。

- 嘴比金箍棒还硬。
- 峡谷里一声古德猫宁。
- 大王今天又要证明自己。

### E. 朋友局甩锅型

功能：适合群唱 / 对唱 / Call and Response。

- 有内鬼啊。
- 谁在演我。
- 这不是我的锅。
- 你先别说话。
- 你看看人家，再看看你。
- 互相举报，互相指责。
- 兄弟们，先把锅分清楚。
- 我们是最好的四个人。

### F. 观众弹幕型

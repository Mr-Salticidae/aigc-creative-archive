# AIGC Creative Archive

这是跳蛛先生的 AIGC 创作归档仓库，用来同步正在进行和已经完成的 AIGC 项目，减少跨设备切换时的进度丢失。

本仓库从 `39_晴枝_花园写真` 开始使用。更早的本地历史项目暂不迁入，避免一次性引入过多旧素材和未整理结构。

## 使用原则

- 一个作品或系列一个独立项目目录。
- 项目目录内必须有 `README.md` 和 `00_项目档案.md`。
- prompt、角色卡、复盘、筛选结论必须提交。
- 精选图和关键过程图可以提交。
- 大量未筛选原图、视频中间文件、下载压缩包默认不提交。

## 同步节奏

每完成一个明确阶段就提交一次，例如：

- 建立角色卡
- 完成一轮 prompt
- 完成一轮出图筛选
- 确认基准图
- 完成发布归档

提交信息建议写成可回溯的动作，例如：

```text
Add Qingzhi R3 moodboard stability analysis
Update Qingzhi oref expansion prompts
Archive selected Qingzhi key images
```

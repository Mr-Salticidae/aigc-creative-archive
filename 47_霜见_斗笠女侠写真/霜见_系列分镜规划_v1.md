# 霜见 · 系列分镜规划 v2

> 2026-06-17 · 一组 8 张,小红书唯美/涨粉向 · 美人主侠为滤镜
> 每张必带装扮签名(斗笠+月白+墨黑+红绳玉佩)+ 冷调胶片底盘;锚定见 [[oref高方差律]]

---

## 设计原则

- **景别不单一**:远/中/近混搭(全特写或全远景都拖质量,见晴枝「叙事让位于美律」反思)。
- **直视镜头 ≤ 2 张**:冷艳孤傲靠"不看镜头"出距离感(见知识库 [[主体不看镜头律]]);只在封面/回眸留 1-2 个直视峰值抓人。
- **借「惊艳三变量」但不照搬**:保留"脸占比大 + 直视峰值",但**光色走冷**(暖金只给客栈那张),不为惊艳牺牲冷艳人设。
- **冷暖对比点缀**:绝大多数冷调,客栈灯下一张暖,作系列里的呼吸。

## 发布排序建议(小红书)

封面 #1(直视抓人)→ #3 抚剑 → #4 回眸 → 新C 侧脸 → 新B 仰拍 → #7 遮脸张力 → 新A 背影 → #6 暖客栈。
节奏:抓人 → 细节 → 再抓人 → 远眺 → 英气 → 神秘 → 离去 → 暖收尾。两张直视(#1#4)被 #3 隔开,不挨着。

## 抽卡顺序建议

1. 先抽 **#1 封面 + #4 回眸**(oref-on,各 8-12 张,过验收挑最佳)——这两张定系列调性。
2. 用 #1#4 的合格脸攒 **moodboard**,再抽 #3 等近景(软锁更稳,见 [[oref高方差律]])。
3. 远景/遮脸(#2#5#7#8)走 oref-off,省事且不挑。
4. #6 客栈单独调暖光 + 降 `--sw`。

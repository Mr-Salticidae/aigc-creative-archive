## 为什么它是合格的风格锚点

| 判据 | 本图 |
|---|---|
| 纯 prompt 产物(非 sref 跑出) | ✅ |
| 形态中性(器物/留白,无人物) | ✅ 斗笠+剑+玉佩+月白绸,无人脸无姿态 |
| 配色完整编码 | ✅ 月白绸 + 墨黑木/剑鞘 + 朱砂红绳,点睛红到位 |
| 胶片质感 | ✅ 冷调低饱和、浅景深、雾竹林虚化,与 R1 同底盘 |
| 构图干净 | ✅ 对角线静物 + 大量留白虚化 |

## 画面拆解(风格指纹构成)

- 主体:编织竹斗笠(后侧立)、黑鞘长剑、盘绕的朱砂红绳、浅色玉佩、堆叠的月白丝绸
- 承托:斑驳深绿/墨黑旧木桌
- 背景:雾气竹林,冷绿灰,强虚化
- 调色:冷调胶片,低饱和,整体青灰带一点土褐暖

## 与 R1 的色温差(记一笔)

R1 人像偏**青灰冷**;本静物的斗笠/红绳偏**土褐暖**一点。两者仍同属"冷调胶片"家族,叠用兼容;若后续整组偏暖,可在场景 prompt 里补 `cool desaturated tones` 往回拉。

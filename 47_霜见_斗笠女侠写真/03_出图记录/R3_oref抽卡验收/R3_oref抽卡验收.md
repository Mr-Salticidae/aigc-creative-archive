# R3 · oref 抽卡验收

> 2026-06-17
> 结论修正:**oref 不是不能用,是高方差**——异常率高,但抽卡 + 验收能稳定捞到好图。

---

## 背景

前面 oref 刷三组效果差(塑料脸/异常),一度判断弃用。但继续抽卡后捞到一张明显合格的(`R3_oref验收_keeper_01.png`),推翻"oref 不可用"的结论。

修正认知:晴枝的「oref 污染美学律」(见 [[晴枝发布复盘_oref污染与工具回归_2026-06-11]])仍成立——oref **倾向**引入磨皮浓妆;但这是概率倾向,不是必然。**抽卡足够多 + 过素颜验收,可以捞到自然皮肤的好图。**

---

## keeper_01 验收

| 验收项 | 结果 |
|---|---|
| 皮肤自然(无磨皮塑料感) | ✅ 通过 |
| 五官骨相(冷艳、杏眼略垂、鹅蛋脸) | ✅ 与 R1 一致 |
| 装扮签名(斗笠/月白/墨黑/红绳) | ✅ 齐 |
| 气质(冷艳孤傲) | ✅ |
| 异常(歪脸/多指/怪手) | ✅ 无 |

→ keeper,可作主视觉候选 / 脸部 moodboard 素材。

---

## 操作结论:oref 高方差律

- oref **能**出好图,但异常率高 → **必须抽卡(一次至少 8-12 张)+ 逐张过"素颜毛孔 + 无异常"验收**,只留通过的。
- 描述锚定(五官词块 + 装扮签名 + sref)是**底座**,无论开不开 oref 都要带——它保证不开 oref 时也成团,开 oref 时减少飘移。
- 美学优先 + 怕塑料 → 可不开 oref(v0.2 路径);要更接近同脸 + 愿意抽卡验收 → 开 oref(v0.1 路径)。
